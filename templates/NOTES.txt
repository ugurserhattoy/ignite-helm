Please note!
{{- if .Values.persistence.enabled }}
Ignite persistence is enabled.

- Check if all pods are running:

kubectl get pods -n {{ .Release.Namespace }} -l app={{ include "ignite.fullname" . }}

- After all pods are running, activate the cluster:

kubectl exec -n {{ .Release.Namespace }} {{ include "ignite.fullname" . }}-0 -- \
  /opt/ignite/apache-ignite-fabric/bin/control.sh --activate

{{- else }}
Ignite is running in in-memory mode. No activation should be needed.

{{- end }}
- You can check the state:

kubectl exec -n {{ .Release.Namespace }} {{ include "ignite.fullname" . }}-0 -- /opt/ignite/apache-ignite-fabric/bin/control.sh --state
